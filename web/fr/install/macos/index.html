macOS

Décompressez l’app, placez-la dans Applications puis ouvrez-la.

La version macOS actuelle est un ZIP qui contient Tutor1on1.app. Le format ZIP est le plus simple à distribuer.

Télécharger le ZIP macOS

Étapes d’installation

  1. Téléchargez Tutor1on1-macos-universal.zip.
  2. Double-cliquez sur le ZIP pour extraire Tutor1on1.app.
  3. Glissez Tutor1on1.app dans /Applications.
  4. Ouvrez-le depuis Applications, Launchpad, Spotlight ou Finder.

Conseils

  • Si Gatekeeper avertit au premier lancement, faites un clic droit sur l’application puis choisissez Ouvrir.
  • Si elle est installée dans Applications, lancez-la avec open -a "Tutor1on1".
  • Le flux normal reste : décompresser, glisser dans Applications puis lancer comme une application.
