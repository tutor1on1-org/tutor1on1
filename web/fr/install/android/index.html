Android

Installez l’APK puis lancez Tutor1on1 comme une application normale.

Tutor1on1 sur Android est distribué comme téléchargement direct d’APK pour une installation manuelle.

Télécharger l’APK Android

Étapes d’installation

  1. Téléchargez Tutor1on1.apk sur votre téléphone ou tablette.
  2. Ouvrez l’APK depuis la liste de téléchargements du navigateur ou depuis votre application Fichiers.
  3. Si Android bloque l’installation, autorisez temporairement l’installation depuis cette source.
  4. Terminez l’installation puis ouvrez Tutor1on1 depuis l’écran de confirmation ou le lanceur.

Lancer plus tard

  • Après installation, ouvrez Tutor1on1 depuis votre lanceur comme n’importe quelle autre application Android.
  • Le fichier APK lui-même n’est plus nécessaire pour le lancement normal après installation.
