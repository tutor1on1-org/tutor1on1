Choisissez votre plateforme

Installez Tutor1on1 correctement pour votre appareil.

Chaque plateforme a son propre format de téléchargement et son propre mode de lancement. Ouvrez la bonne page au lieu de vous fier seulement au nom du fichier.

Windows

ZIP portable

Téléchargez le ZIP, extrayez-le dans un dossier normal puis lancez tutor1on1.exe.

Étapes Windows
Android

Téléchargement APK

Téléchargez l’APK, autorisez l’installation depuis cette source puis ouvrez Tutor1on1 depuis le lanceur.

Étapes Android

Hôte de téléchargement

  • Les pages du site sont servies depuis www.tutor1on1.org.
  • Les fichiers d’installation sont servis depuis api.tutor1on1.org/downloads/.
  • Le site et les artefacts téléchargeables vivent sur des hôtes différents.
