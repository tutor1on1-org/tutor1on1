Windows

Téléchargez le ZIP, extrayez-le puis lancez l’application.

Tutor1on1 sur Windows est actuellement distribué sous forme de ZIP portable. Il n’y a pas d’installateur séparé.

Télécharger le ZIP Windows

Étapes d’installation

  1. Téléchargez Tutor1on1.zip.
  2. Extrayez le ZIP dans un dossier normal comme Downloads\Tutor1on1 ou C:\Apps\Tutor1on1.
  3. Ouvrez le dossier extrait et lancez Tutor1on1.exe.
  4. Si Windows SmartScreen apparaît, continuez seulement si vous faites confiance à la source.

Lancer plus tard

  • Vous pourrez relancer l’application en ouvrant Tutor1on1.exe depuis le dossier extrait.
  • Si vous voulez un raccourci, créez-le manuellement après extraction.
  • N’exécutez pas l’application directement depuis le visualiseur ZIP.
