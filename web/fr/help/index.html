Demarrage rapide et API config

Demarrage rapide et API config

Installez l app, connectez vous comme professeur ou eleve, puis comprenez le role de chaque champ de config.

Les professeurs creent ou chargent un cours, televersent un bundle valide, approuvent les inscriptions et suivent progres et erreurs par eleve.

Les eleves entrent dans un cours approuve, etudient avec Learn et Review, recoivent des exercices adaptes et utilisent le carnet erreurs.

API config

  • Provider: preset du service API, comme OpenAI ou SiliconFlow.
  • Base URL: endpoint compatible OpenAI. Test API key appelle baseUrl + /models.
  • API key: cle du fournisseur gardee localement dans secure storage pour ce base URL.
  • Text model: conversation, guidage, feedback et evaluation Review.
  • TTS/STT models: modeles optionnels de voix et reconnaissance.
  • Reasoning effort: reglage optionnel pour fournisseurs compatibles.
  • Test API key: verifie acces et actualise les modeles. Ne telecharge pas de modele et ne selectionne pas automatiquement un nouveau modele.

Aide

Si la liste semble ancienne, relancez Test API key. Si un eleve ne peut pas entrer dans un cours, verifiez approbation et bundle.
