macOS

App entpacken, nach Applications verschieben und dann öffnen.

Die aktuelle macOS-Version ist eine ZIP-Datei mit Tutor1on1.app. ZIP ist das praktische Downloadformat für diesen Build.

macOS-ZIP herunterladen

Installationsschritte

  1. Tutor1on1-macos-universal.zip herunterladen.
  2. Doppelklicken Sie auf die ZIP-Datei, um Tutor1on1.app zu entpacken.
  3. Ziehen Sie Tutor1on1.app nach /Applications.
  4. Öffnen Sie sie aus Applications, Launchpad, Spotlight oder Finder.

Hinweise

  • Wenn Gatekeeper beim ersten Start warnt, klicken Sie die App mit rechts an und wählen Sie Öffnen.
  • Wenn die App in Applications liegt, starten Sie sie mit open -a "Tutor1on1".
  • Der normale Ablauf bleibt: entpacken, nach Applications ziehen und als App starten.
