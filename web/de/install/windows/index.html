StartInstallieren

Windows

ZIP herunterladen, entpacken und dann die App starten.

Tutor1on1 für Windows wird derzeit als portables ZIP verteilt. Es gibt keinen separaten Installer.

Windows-ZIP herunterladen

Installationsschritte

  1. Tutor1on1-1.0.11.zip herunterladen.
  2. Die ZIP-Datei in einen normalen Ordner wie Downloads\Tutor1on1 oder C:\Apps\Tutor1on1 entpacken.
  3. Den entpackten Ordner öffnen und tutor1on1.exe starten.
  4. Falls Windows SmartScreen erscheint, fahren Sie nur fort, wenn Sie der Quelle vertrauen.

Später starten

  • Sie können die App später erneut starten, indem Sie tutor1on1.exe im entpackten Ordner öffnen.
  • Wenn Sie eine Verknüpfung möchten, erstellen Sie sie nach dem Entpacken manuell.
  • Nicht direkt aus dem ZIP-Viewer starten.
