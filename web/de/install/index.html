Plattform wählen

Installieren Sie Tutor1on1 passend für Ihr Gerät.

Jede Plattform hat ihr eigenes Downloadformat und Startverhalten. Öffnen Sie die passende Seite, statt nur anhand des Dateinamens zu raten.

Windows

Portables ZIP-Build

Laden Sie die ZIP-Datei herunter, entpacken Sie sie in einen normalen Ordner und starten Sie Tutor1on1.exe.

Windows-Schritte
Android

APK-Download

Laden Sie die APK herunter, erlauben Sie die Installation aus dieser Quelle und öffnen Sie dann Tutor1on1 im App-Drawer.

Android-Schritte

Download-Host

  • Die Webseiten werden von www.tutor1on1.org ausgeliefert.
  • Installationsdateien werden von api.tutor1on1.org/downloads/ ausgeliefert.
  • Website und herunterladbare Artefakte liegen auf unterschiedlichen Hosts.
