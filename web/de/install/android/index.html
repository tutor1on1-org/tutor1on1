Android

APK installieren und Tutor1on1 wie eine normale App starten.

Tutor1on1 für Android wird als direkter APK-Download für manuelle Installation verteilt.

Android-APK herunterladen

Installationsschritte

  1. family_teacher.apk auf Ihr Telefon oder Tablet herunterladen.
  2. Die APK aus der Downloadliste des Browsers oder aus Ihrer Dateien-App öffnen.
  3. Falls Android die Installation blockiert, die Installation aus dieser Quelle vorübergehend erlauben.
  4. Installation abschließen und Tutor1on1 dann aus dem Bestätigungsbildschirm oder dem App-Drawer öffnen.

Später starten

  • Nach der Installation öffnen Sie Tutor1on1 wie jede andere Android-App über Ihren Launcher.
  • Die APK-Datei selbst wird nach abgeschlossener Installation für den normalen Start nicht mehr benötigt.
