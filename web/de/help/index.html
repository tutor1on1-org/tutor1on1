So verwenden Sie Tutor1on1

Kurzhilfe für Lehrkräfte und Lernende.

Tutor1on1 ist für Windows und Android verfügbar. Lehrkräfte veröffentlichen strukturierte Kurse. Lernende schreiben sich ein, laden freigegebene Inhalte herunter und lernen in geführten Sitzungen.

Erste Schritte

  1. Öffnen Sie zuerst die Installationsanleitung und wählen Sie Windows oder Android.
  2. Wählen Sie bei Bedarf eine Sprache.
  3. Öffnen Sie Einstellungen und speichern Sie die benötigten API keys, bevor Sie Tutorfunktionen nutzen.
  4. Melden Sie sich an oder registrieren Sie sich als Lehrkraft bzw. Lernender auf der Willkommensseite.

Schnellstart für Lehrkräfte

  1. Als Lehrkraft anmelden.
  2. Eine lokale Kursversion erstellen oder laden.
  3. Das validierte Bundle in den Marketplace hochladen.
  4. Einschreibungsanfragen freigeben und Fortschritt prüfen.

Schnellstart für Lernende

  1. Als Lernender anmelden.
  2. Im Marketplace einen Kurs suchen und Einschreibung anfragen.
  3. Nach Freigabe das Bundle herunterladen.
  4. Den Kurs öffnen und Lernen oder Wiederholen starten.

Synchronisation und häufige Probleme

  • Die Startseiten synchronisieren im Hintergrund, wenn das Konto mit dem Server verknüpft ist.
  • Einschreibungsstatus, Sitzungen und Fortschritt können zwischen Geräten wechseln.
  • Wenn Anmeldung oder Kursladen fehlschlägt, prüfen Sie zuerst, ob die App api.tutor1on1.org erreichen kann.
