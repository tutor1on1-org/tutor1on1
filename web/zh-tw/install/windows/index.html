Windows

下載 ZIP，解壓，然後執行應用。

Tutor1on1 Windows 版目前以可攜式 ZIP 發布，沒有獨立安裝程式。

下載 Windows ZIP

安裝步驟

  1. 下載 Tutor1on1.zip。
  2. 把 ZIP 解壓到一般資料夾，例如 Downloads\Tutor1on1 或 C:\Apps\Tutor1on1。
  3. 開啟解壓後的資料夾並執行 family_teacher.exe。
  4. 若出現 Windows SmartScreen，請先確認來源可信。

之後如何執行

  • 之後可直接從解壓後的資料夾開啟 family_teacher.exe。
  • 若需要捷徑，請在解壓後手動建立。
  • 不要直接在 ZIP 檢視器中執行。
