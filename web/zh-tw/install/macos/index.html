macOS

解壓應用，移到 Applications，再開啟。

目前 macOS 版本以包含 Tutor1on1.app 的 ZIP 發布。ZIP 會比裸 .app 更適合下載分發。

下載 macOS ZIP

安裝步驟

  1. 下載 Tutor1on1-macos-universal.zip。
  2. 雙擊 ZIP，解壓出 Tutor1on1.app。
  3. 把 Tutor1on1.app 拖到 /Applications。
  4. 從 Applications、Launchpad、Spotlight 或 Finder 開啟它。

補充提示

  • 若首次啟動時 Gatekeeper 警告，請在 Finder 中右鍵應用並選擇 打開。
  • 若已安裝到 Applications，可用 open -a "Tutor1on1" 啟動。
  • 一般使用流程仍然是：解壓，拖到 Applications，然後執行。
