Android

安裝 APK，然後像一般應用一樣啟動 Tutor1on1。

Tutor1on1 Android 版透過 APK 直接提供，需要手動安裝。

下載 Android APK

安裝步驟

  1. 把 Tutor1on1-1.0.11.apk 下載到手機或平板。
  2. 從瀏覽器下載列表或檔案應用中開啟 APK。
  3. 若 Android 阻擋安裝，暫時允許該來源安裝。
  4. 完成安裝後，從確認畫面或應用程式抽屜開啟 Tutor1on1。

之後如何執行

  • 安裝完成後，像一般 Android 應用一樣從啟動器中開啟 Tutor1on1。
  • 安裝完成後，APK 檔本身不再是日常啟動所必需。
