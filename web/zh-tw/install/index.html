選擇你的平台

依照你的裝置選擇正確的安裝方式。

不同平台的下載格式與啟動方式不同。請先進入對應平台頁面，再依步驟安裝。

Windows

可攜式 ZIP

下載 ZIP，解壓到一般資料夾後執行 family_teacher.exe。

Windows 步驟
Android

APK 下載

下載 APK，允許目前來源安裝，然後從應用程式列表開啟 Tutor1on1。

Android 步驟
macOS

包含 Tutor1on1.app 的 ZIP

下載 ZIP，解壓後把 Tutor1on1.app 拖到 /Applications，再正常開啟。

macOS 步驟

下載主機

  • 網站頁面由 www.tutor1on1.org 提供。
  • 安裝檔由 api.tutor1on1.org/downloads/ 提供。
  • 站點頁面與可下載檔案位於不同主機，這是刻意設計。
