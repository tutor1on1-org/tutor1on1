快速開始與 API config

快速開始與 API config

安裝應用，按老師或學生身份登入，並理解 config 中每個欄位控制什麼。

老師建立或載入課程，上傳已校驗 bundle，審批報名，並查看每位學生的進度與錯題模式。

學生報名已批准課程，透過 Learn 與 Review 學習，獲得針對練習，並用錯題集複習薄弱點。

API config

  • Provider：API 服務預設，例如 OpenAI 或 SiliconFlow。
  • Base URL：OpenAI-compatible endpoint。Test API key 會請求 baseUrl + /models。
  • API key：服務商 key，按 base URL 存在本機 secure storage。
  • Text model：用於 tutor 對話、講解、回饋和 Review 判題。
  • TTS/STT models：可選語音輸出和語音辨識模型。
  • Reasoning effort：部分服務商支援的可選 reasoning 設定。
  • Test API key：驗證訪問並刷新模型下拉列表；不會下載模型，也不會自動選擇新模型。

說明

如果模型列表不新，重新點擊 Test API key。如果學生不能進入課程，確認報名已批准且 bundle 已下載。如果進度不同步，等待同步完成或手動刷新。
