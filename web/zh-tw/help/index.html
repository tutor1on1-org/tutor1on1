如何使用 Tutor1on1

給老師與學生的快速說明。

Tutor1on1 可在 Windows 與 Android 上使用。老師發布結構化課程，學生報名、下載已核准內容，並透過引導式會話學習。

開始使用

  1. 先開啟安裝指南，選擇 Windows 或 Android。
  2. 如有需要，先切換語言。
  3. 在設定中填入並儲存所需 API key，再開始使用輔導功能。
  4. 在歡迎頁登入，或註冊成老師、學生。

老師快速開始

  1. 以老師身分登入。
  2. 建立或載入本地課程版本。
  3. 把已驗證的 bundle 上傳到市場。
  4. 審批學生的報名請求並查看進度。

學生快速開始

  1. 以學生身分登入。
  2. 瀏覽市場並申請加入課程。
  3. 等待老師核准並下載課程包。
  4. 開啟課程並開始學習或複習。

同步與常見問題

  • 帳號連結到伺服器後，老師與學生首頁都會在背景同步。
  • 報名狀態、會話與進度可以跨裝置流轉。
  • 若登入或課程載入失敗，先檢查應用是否能連到 api.tutor1on1.org。
