Quick start

Use Tutor1on1 by role.

Teachers publish and manage courses. Students request enrollment, download approved content, and study through guided sessions.

Install first Teacher flow Student flow

Teacher flow

Publish and manage courses.

  1. Log in as a teacher or register a teacher account.
  2. Create or load a local course version.
  3. Upload the validated bundle to the marketplace.
  4. Approve student enrollment requests.
  5. Review student progress and synced sessions from the teacher home page.

Student flow

Enroll and study.

  1. Log in as a student or register a student account.
  2. Open the marketplace and request enrollment in a course.
  3. Wait for teacher approval.
  4. Download the approved course bundle.
  5. Open the course and start a learn or review session.

Sign in

Use the welcome page tabs to log in, register as a teacher, or register as a student.

Marketplace

Course discovery

Teachers publish course listings. Students browse, request enrollment, and download after approval.

Sessions

Learn, review, summarize

Use Learn for new material, Review for recall practice, and Summary when the session should update mastery.

Sync

Continuity

Teacher and student home pages sync account-linked enrollment, sessions, and progress in the background.

Common problems

  • If login or course loading fails, check that the app can reach api.tutor1on1.org.
  • If a student cannot enter a course, confirm that enrollment was approved and the bundle was downloaded.
  • If progress looks stale across devices, wait for sync to finish or run a manual refresh.
  • If a session is interrupted, reopen the same course and continue the existing session.
