macOS

アプリを解凍し、Applications へ移してから開く。

現在の macOS 版は Tutor1on1.app を含む ZIP です。ZIP の方がダウンロード配布に適しています。

macOS ZIP をダウンロード

インストール手順

  1. Tutor1on1-macos-universal.zip をダウンロードします。
  2. ZIP をダブルクリックして Tutor1on1.app を展開します。
  3. Tutor1on1.app を /Applications にドラッグします。
  4. Applications、Launchpad、Spotlight、または Finder から開きます。

補足

  • 初回起動で Gatekeeper の警告が出た場合は、Finder でアプリを右クリックして 開く を選んでください。
  • Applications に入っていれば open -a "Tutor1on1" で起動できます。
  • 通常の手順は、解凍して Applications に移し、アプリとして起動することです。
