プラットフォームを選ぶ

端末に合った方法で Tutor1on1 をインストール。

プラットフォームごとにダウンロード形式と起動手順が異なります。ファイル名だけで判断せず、下の案内ページを開いてください。

Windows

ポータブル ZIP

ZIP をダウンロードし、通常のフォルダへ展開して family_teacher.exe を実行します。

Windows 手順
Android

APK ダウンロード

APK をダウンロードし、その提供元からのインストールを許可してからアプリ一覧で Tutor1on1 を開きます。

Android 手順
macOS

Tutor1on1.app を含む ZIP

ZIP をダウンロードし、解凍してから Tutor1on1.app を /Applications へ移動し、通常どおり起動します。

macOS 手順

配布ホスト

  • Web ページは www.tutor1on1.org から配信されます。
  • インストールファイルは api.tutor1on1.org/downloads/ から配信されます。
  • サイト本体とダウンロード成果物は別のホストに置かれています。
