ホームインストール

Windows

ZIP をダウンロードし、展開してからアプリを実行。

Tutor1on1 の Windows 版は現在ポータブル ZIP として配布されています。個別のインストーラーはありません。

Windows ZIP をダウンロード

インストール手順

  1. family_teacher.zip をダウンロードします。
  2. ZIP を Downloads\Tutor1on1 や C:\Apps\Tutor1on1 のような通常フォルダへ展開します。
  3. 展開したフォルダを開き、family_teacher.exe を実行します。
  4. Windows SmartScreen が表示された場合は、配布元を確認したうえで続行してください。

あとで起動する

  • 以後は展開済みフォルダの family_teacher.exe を開けば再起動できます。
  • ショートカットが必要なら、展開後に手動で作成してください。
  • ZIP ビューアー内から直接実行せず、必ず先に展開してください。
