ホームインストール

Android

APK を入れて、通常のアプリのように Tutor1on1 を起動。

Tutor1on1 の Android 版は直接 APK を配布しており、手動インストールが必要です。

Android APK をダウンロード

インストール手順

  1. Tutor1on1-1.0.10.apk をスマートフォンまたはタブレットにダウンロードします。
  2. ブラウザのダウンロード一覧、または Files アプリから APK を開きます。
  3. Android がインストールをブロックした場合は、その提供元からのインストールを一時的に許可します。
  4. インストール完了後、確認画面またはアプリ一覧から Tutor1on1 を開きます。

あとで起動する

  • インストール後は、通常の Android アプリと同じようにランチャーから Tutor1on1 を開きます。
  • 通常起動には、インストール完了後の APK ファイル自体は不要です。
