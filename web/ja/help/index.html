Tutor1on1 の使い方

教師と学習者のためのクイックガイド。

Tutor1on1 は Windows と Android で利用できます。教師は構造化コースを公開し、学習者は受講して承認済みコンテンツをダウンロードし、ガイド付きセッションで学びます。

始め方

  1. インストールガイドを開き、Windows または Androidを選びます。
  2. 必要なら最初に言語を切り替えます。
  3. 設定で必要な API key を保存してからチューター機能を使います。
  4. ようこそページからログインするか、教師または学習者として登録します。

教師クイックスタート

  1. 教師としてログインします。
  2. ローカルのコース版を作成または読み込みます。
  3. 検証済み bundle をマーケットプレイスへアップロードします。
  4. 学習者の受講申請を承認し、進捗を確認します。

学習者クイックスタート

  1. 学習者としてログインします。
  2. マーケットプレイスでコースを探して受講申請します。
  3. 承認後にコース bundle をダウンロードします。
  4. コースを開いて学習または復習を始めます。

同期とよくある問題

  • アカウントがサーバーに接続されている場合、ホームはバックグラウンドで同期します。
  • 受講状態、セッション、進捗は端末間で引き継げます。
  • ログインやコース読み込みに失敗したら、まずアプリが api.tutor1on1.org に到達できるか確認してください。

まずアプリが必要ですか？
