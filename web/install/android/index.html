Android

Install the APK, then launch Tutor1on1 like a normal app.

Tutor1on1 on Android is distributed as a direct APK download for manual installation.

Download Android APK

Install steps

  1. Download family_teacher.apk to your phone or tablet.
  2. Open the APK from the browser downloads list or your Files app.
  3. If Android blocks the install, temporarily allow installation from that source.
  4. Finish the install, then open Tutor1on1 from the confirmation screen or app drawer.

Run later

  • After installation, open Tutor1on1 from your launcher like any other Android app.
  • The APK file itself is no longer needed for normal launching after install completes.
