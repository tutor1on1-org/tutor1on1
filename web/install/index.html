Choose your platform

Install Tutor1on1 the right way for your device.

Each platform has its own download format and launch flow. Open the platform page below instead of guessing from the file name alone.

Windows

Portable ZIP build

Download the ZIP, extract it to a normal folder, and run Tutor1on1.exe.

Windows steps
Android

APK download

Download the APK, allow installation from the selected source, then open Tutor1on1 from the app drawer.

Android steps

Download host

  • Website pages are served from www.tutor1on1.org.
  • Install files are served from api.tutor1on1.org/downloads/.
  • That split is intentional: the site and the downloadable artifacts live on different virtual hosts.
