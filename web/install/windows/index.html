Windows

Download the ZIP, extract it, then run the app.

Tutor1on1 on Windows is distributed as a portable ZIP. Extract before launching so the app can access its bundled files.

Download Windows ZIP

Install steps

1

Download

Save Tutor1on1.zip from the download button above.

2

Extract

Extract the ZIP to a normal folder such as Downloads\Tutor1on1 or C:\Apps\Tutor1on1.

3

Run

Open the extracted folder and launch tutor1on1.exe.

Notes

  • If Windows SmartScreen appears, review the prompt and continue only if you trust the build source.
  • Do not run Tutor1on1 directly from inside the ZIP viewer. Extract first.
  • Create a shortcut manually after extraction if you want one.
