Windows

Download the ZIP, extract it, then run the app.

Tutor1on1 on Windows is currently distributed as a portable ZIP. There is no separate installer.

Download Windows ZIP

Install steps

  1. Download Tutor1on1.zip.
  2. Extract the ZIP to a normal folder such as Downloads\Tutor1on1 or C:\Apps\Tutor1on1.
  3. Open the extracted folder and run tutor1on1.exe.
  4. If Windows SmartScreen appears, review the prompt and continue only if you trust the build source.

Run later

  • You can run the app again by opening tutor1on1.exe from the extracted folder.
  • If you want a shortcut, create one manually after extraction.
  • Do not run it directly from inside the ZIP viewer. Extract first.
