macOS

Unzip the app, move it to Applications, then open it.

The current macOS release is a ZIP that contains Tutor1on1.app. A raw .app is a directory bundle, so it should not be distributed as a bare download file.

Download macOS ZIP

Why ZIP and not raw .app?

  • A macOS .app is really a folder bundle, not a single flat file.
  • ZIP preserves the full app bundle as one downloadable artifact.
  • A .dmg is also valid, but ZIP is the simpler current release format.

Install steps

  1. Download Tutor1on1-macos-universal.zip.
  2. Double-click the ZIP to extract Tutor1on1.app.
  3. Drag Tutor1on1.app into /Applications.
  4. Open it from Applications, Launchpad, Spotlight, or Finder.
  5. If Gatekeeper warns on first launch, use Finder, right-click the app, then choose Open.

Run from another app or script

  • If installed in Applications, launch it with open -a "Tutor1on1".
  • If not installed yet, launch the bundle by path with open "/path/to/Tutor1on1.app".
  • The normal user flow is still: unzip, drag to Applications, then run it as an app.
