Product workflow

Built around the course lifecycle, not a generic chat box.

1

Teachers publish structured material

Course bundles keep lectures, questions, review prompts, and version metadata together.

2

Students request enrollment

The marketplace handles discovery, approval, and approved bundle downloads.

3

Sessions preserve progress

Guided tutor work produces durable progress that can sync back to the account.

For teachers

Publish once, manage continuously.

Prepare local course versions, upload validated bundles, approve enrollments, and review student progress.

Browse the catalog, request enrollment, download approved bundles, and continue learn or review sessions.

Keep the learning state connected.
