플랫폼 선택

기기에 맞는 방식으로 Tutor1on1을 설치하세요.

플랫폼마다 다운로드 형식과 실행 방식이 다릅니다. 파일 이름만 보고 추측하지 말고 아래 플랫폼 페이지를 여세요.

Windows

포터블 ZIP

ZIP을 내려받아 일반 폴더에 압축을 풀고 family_teacher.exe 를 실행하세요.

Windows 단계
Android

APK 다운로드

APK를 내려받고 해당 출처의 설치를 허용한 뒤 앱 서랍에서 Tutor1on1을 여세요.

Android 단계
macOS

Tutor1on1.app 이 포함된 ZIP

ZIP을 내려받아 압축을 풀고 Tutor1on1.app 을 /Applications 로 옮긴 뒤 실행하세요.

macOS 단계

배포 호스트

  • 웹 페이지는 www.tutor1on1.org 에서 제공됩니다.
  • 설치 파일은 api.tutor1on1.org/downloads/ 에서 제공됩니다.
  • 사이트와 다운로드 산출물은 서로 다른 호스트에 있습니다.
