Windows

ZIP을 내려받고 압축을 푼 뒤 앱을 실행하세요.

Tutor1on1 Windows 버전은 현재 포터블 ZIP으로 배포되며 별도 설치 프로그램은 없습니다.

Windows ZIP 다운로드

설치 단계

  1. Tutor1on1.zip 을 내려받습니다.
  2. ZIP을 Downloads\Tutor1on1 또는 C:\Apps\Tutor1on1 같은 일반 폴더에 풉니다.
  3. 압축을 푼 폴더를 열고 family_teacher.exe 를 실행합니다.
  4. Windows SmartScreen이 나타나면 빌드 출처를 확인한 뒤 계속 진행하세요.

나중에 다시 실행하기

  • 다음부터는 압축을 푼 폴더의 family_teacher.exe 를 열어 실행하면 됩니다.
  • 바로가기가 필요하면 압축 해제 후 수동으로 만드세요.
  • ZIP 보기 창 안에서 직접 실행하지 말고 먼저 압축을 푸세요.
