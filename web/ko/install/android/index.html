Android

APK를 설치한 뒤 일반 앱처럼 Tutor1on1을 실행하세요.

Tutor1on1 Android 버전은 직접 APK로 배포되며 수동 설치가 필요합니다.

Android APK 다운로드

설치 단계

  1. Tutor1on1-1.0.12.apk 를 휴대폰이나 태블릿에 내려받습니다.
  2. 브라우저 다운로드 목록이나 파일 앱에서 APK를 엽니다.
  3. Android가 설치를 막으면 해당 출처 설치를 잠시 허용합니다.
  4. 설치가 끝나면 확인 화면이나 앱 서랍에서 Tutor1on1을 엽니다.

나중에 다시 실행하기

  • 설치 후에는 다른 Android 앱처럼 실행기에서 Tutor1on1을 열면 됩니다.
  • 설치가 끝나면 일반 실행에는 APK 파일 자체가 더 이상 필요하지 않습니다.
