macOS

앱을 압축 해제하고 Applications로 옮긴 뒤 여세요.

현재 macOS 릴리스는 Tutor1on1.app 이 들어 있는 ZIP입니다. ZIP 형식이 다운로드 배포에 더 적합합니다.

macOS ZIP 다운로드

설치 단계

  1. Tutor1on1-macos-universal.zip 을 내려받습니다.
  2. ZIP을 두 번 클릭해 Tutor1on1.app 을 풉니다.
  3. Tutor1on1.app 을 /Applications 로 드래그합니다.
  4. Applications, Launchpad, Spotlight 또는 Finder에서 엽니다.

추가 팁

  • 처음 실행 때 Gatekeeper 경고가 나오면 Finder에서 앱을 우클릭하고 열기 를 선택하세요.
  • Applications에 설치했다면 open -a "Tutor1on1" 으로 실행할 수 있습니다.
  • 일반 사용자 흐름은 압축 해제 후 Applications로 옮기고 앱처럼 실행하는 것입니다.
