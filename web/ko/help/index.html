빠른 시작 및 API config

앱을 설치하고 교사 또는 학생으로 로그인한 뒤 config의 각 항목이 무엇을 제어하는지 확인합니다.

교사는 코스를 만들거나 불러오고 검증된 bundle을 업로드하며 등록 요청을 승인하고 학생별 진도와 오답 패턴을 확인합니다.

학생은 승인된 코스에서 Learn 및 Review를 진행하고 맞춤 연습을 받으며 오답 노트로 약점을 복습합니다.

API config

  • Provider: OpenAI 또는 SiliconFlow 같은 API 서비스 프리셋입니다.
  • Base URL: OpenAI-compatible endpoint입니다. Test API key는 baseUrl + /models를 호출합니다.
  • API key: 제공자 key이며 base URL별로 로컬 secure storage에 저장합니다.
  • Text model: 튜터 대화, 설명, 피드백, Review 채점에 사용합니다.
  • TTS/STT models: 선택 음성 출력 및 음성 인식 모델입니다.
  • Reasoning effort: 지원 provider용 선택 설정입니다.
  • Test API key: 접근을 확인하고 모델 목록을 갱신합니다. 모델을 다운로드하지 않고 새 모델을 자동 선택하지 않습니다.

도움말

모델 목록이 오래되었으면 Test API key를 다시 실행하세요. 학생이 코스에 들어가지 못하면 승인과 bundle 다운로드를 확인하세요.
