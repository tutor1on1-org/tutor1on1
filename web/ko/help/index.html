Tutor1on1 사용법

교사와 학생을 위한 빠른 도움말.

Tutor1on1은 Windows, Android, macOS에서 사용할 수 있습니다. 교사는 구조화된 코스를 게시하고, 학생은 등록 후 승인된 콘텐츠를 내려받아 안내형 세션으로 학습합니다.

시작하기

  1. 설치 가이드를 열고 Windows, Android, macOS 중 하나를 고릅니다.
  2. 필요하면 먼저 언어를 바꿉니다.
  3. 설정에서 필요한 API key를 저장한 뒤 튜터 기능을 사용하세요.
  4. 환영 페이지에서 로그인하거나 교사 또는 학생으로 등록하세요.

교사 빠른 시작

  1. 교사로 로그인합니다.
  2. 로컬 코스 버전을 만들거나 불러옵니다.
  3. 검증된 bundle을 마켓플레이스에 업로드합니다.
  4. 학생 등록 요청을 승인하고 진행 상황을 확인합니다.

학생 빠른 시작

  1. 학생으로 로그인합니다.
  2. 마켓플레이스에서 코스를 찾아 등록을 요청합니다.
  3. 승인 후 코스 bundle을 내려받습니다.
  4. 코스를 열고 학습 또는 복습을 시작합니다.

동기화와 자주 겪는 문제

  • 계정이 서버와 연결되면 홈이 백그라운드에서 동기화됩니다.
  • 등록 상태, 세션, 진행이 기기 사이를 이동할 수 있습니다.
  • 로그인이나 코스 불러오기에 실패하면 먼저 앱이 api.tutor1on1.org 에 연결되는지 확인하세요.
