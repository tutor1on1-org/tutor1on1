如何使用 Tutor1on1

给老师和学生的快速帮助页。

Tutor1on1 是桌面端应用。老师发布结构化课程，学生报名、下载已批准内容，并通过引导式会话学习。

1. 首次打开

  1. 打开桌面应用。
  2. 如有需要先切换语言。
  3. 先打开设置页，填写需要的 API key，并点击保存，然后再开始使用 tutor 相关功能。
  4. 在欢迎页使用标签页进行登录、老师注册或学生注册。
  5. 老师注册会填写资料并选择学科标签，学生注册流程更简单。

Marketplace 与课程

  • Marketplace 入口是应用右上角附近的那个小图标。
  • 老师可以在那里处理远端课程发布和相关 marketplace 流程。
  • 学生可以在那里浏览课程、申请报名，并在批准后开始下载流程。

老师快速开始

发布并管理课程。

  1. 以老师身份登录。
  2. 创建或加载本地课程版本。
  3. 将已校验的 bundle 上传到 marketplace。
  4. 审批学生报名请求。
  5. 在老师主页查看学生进度和同步过来的会话。

学生快速开始

报名并开始学习。

  1. 以学生身份登录。
  2. 浏览 marketplace 并申请课程报名。
  3. 等待老师批准。
  4. 下载已批准的课程 bundle。
  5. 打开课程并开始学习会话。

2. 学习会话模式

Learn

Learn 用于学习新内容，由 tutor 带着你走完整个知识点。

Review

Review 用于做题和回忆，强化已经接触过的知识点。

Summary

Summary 用于总结本次会话，并更新当前知识点的掌握结果。

3. 同步与连续性

  • 当账号连接到服务器时，老师主页和学生主页会在后台自动同步。
  • 报名状态、会话和进度可以在多设备之间延续。
  • 学生端还提供强制从服务器拉取的选项，用于刷新服务器副本。

4. 常见问题

  • 如果登录或课程加载失败，先确认应用能连到 `api.tutor1on1.org`。
  • 如果学生无法进入课程，先检查报名是否已批准，以及课程 bundle 是否下载成功。
  • 如果多设备进度不同步，等待同步完成，或手动刷新一次。
  • 如果学习会话中断，重新打开同一门课程并继续已有会话。
