快速开始

按角色使用 Tutor1on1。

老师发布并管理课程；学生申请报名、下载已批准内容，并通过引导式会话学习。

先安装 老师流程 学生流程

老师流程

发布并管理课程。

  1. 以老师身份登录，或注册老师账号。
  2. 创建或加载本地课程版本。
  3. 将已校验的 bundle 上传到 marketplace。
  4. 审批学生报名请求。
  5. 在老师主页查看学生进度和同步过来的会话。

学生流程

报名并开始学习。

  1. 以学生身份登录，或注册学生账号。
  2. 打开 marketplace，申请报名课程。
  3. 等待老师批准。
  4. 下载已批准的课程 bundle。
  5. 打开课程，开始 Learn 或 Review 会话。

首次打开检查项

1

安装

在安装页面选择 Windows 或 Android，然后启动应用。

2

配置

打开设置页，填写 tutor 功能需要的 API 配置，并点击保存。

3

登录

在欢迎页使用标签页登录，或注册老师/学生账号。

Marketplace

课程发现

老师发布课程展示；学生浏览课程、申请报名，并在批准后下载。

Sessions

学习、复习、总结

Learn 用于新内容，Review 用于回忆练习，Summary 用于更新掌握结果。

Sync

连续学习

老师和学生主页会同步已连接账号的报名、会话和进度。

常见问题

  • 如果登录或课程加载失败，先确认应用能连到 api.tutor1on1.org。
  • 如果学生无法进入课程，先确认报名已批准，并且 bundle 已下载。
  • 如果多设备进度不同步，等待同步完成，或手动刷新一次。
  • 如果学习会话中断，重新打开同一门课程并继续已有会话。
