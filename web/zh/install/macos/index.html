macOS

解压应用，拖到 Applications，再打开。

当前 macOS 版本以 ZIP 形式发布，ZIP 内包含 Tutor1on1.app。 裸 .app 实际上是目录 bundle，不适合直接作为单个下载文件发布。

下载 macOS ZIP

为什么用 ZIP，不直接放 .app？

  • macOS 的 .app 本质上是一个目录 bundle，不是单个扁平文件。
  • ZIP 可以把整个 app bundle 作为一个完整下载物保存下来。
  • .dmg 也可以，但 ZIP 是当前更简单的发布格式。

安装步骤

  1. 下载 Tutor1on1-macos-universal.zip。
  2. 双击 ZIP，解压出 Tutor1on1.app。
  3. 把 Tutor1on1.app 拖到 /Applications。
  4. 从 Applications、Launchpad、Spotlight 或 Finder 中打开它。
  5. 如果首次启动遇到 Gatekeeper 提示，可在 Finder 中右键应用，再选择打开。

从其他应用或脚本启动

  • 如果已经安装到 Applications，可用 open -a "Tutor1on1" 启动。
  • 如果还未安装，也可以按路径启动 bundle：open "/path/to/Tutor1on1.app"。
  • 正常用户流程仍然是：解压，拖到 Applications，然后像普通应用一样运行。
