Android

安装 APK，然后从应用列表打开 Tutor1on1。

Android 版 Tutor1on1 通过 APK 直接下载，需要手动安装。

下载 Android APK

安装步骤

1

下载

把 Tutor1on1.apk 保存到手机或平板。

2

允许安装

从浏览器下载列表或文件管理器打开 APK。如果 Android 拦截安装，请允许当前来源安装。

3

启动

安装完成后，从确认页面或应用列表打开 Tutor1on1。

注意

  • APK 文件只用于安装。安装完成后，正常从应用图标启动。
  • 如果安装被拦截，请确认允许的是刚才下载 APK 的浏览器或文件管理器。
