Android

安装 APK，然后像普通应用一样打开 Tutor1on1。

Android 版目前以 APK 直链形式发布，采用手动安装流程。

下载 Android APK

安装步骤

  1. 下载 family_teacher.apk 到手机或平板。
  2. 从浏览器下载列表或文件管理器中打开 APK。
  3. 如果 Android 阻止安装，临时允许该来源安装应用。
  4. 安装完成后，从确认页或应用抽屉中打开 Tutor1on1。

以后如何运行

  • 安装完成后，像普通 Android 应用一样从启动器打开 Tutor1on1。
  • 安装完成后，APK 文件本身不再是正常启动所必需的。
