选择平台

根据设备选择正确的安装方式。

不同平台的下载格式和启动方式不同。请先进入对应平台页面，再按步骤安装，不要只看文件名猜测。

Windows

便携 ZIP

下载 ZIP，解压后运行 tutor1on1.exe。

Windows 步骤
Android

APK 安装包

下载 APK，允许当前来源安装，然后从应用列表打开 Tutor1on1。

Android 步骤

下载主机

  • 网站页面由 www.tutor1on1.org 提供。
  • 安装文件由 api.tutor1on1.org/downloads/ 提供。
  • 这是刻意分开的：站点页面与可下载文件位于不同虚拟主机。
