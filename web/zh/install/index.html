选择平台

在 Windows 或 Android 上安装 Tutor1on1。

先下载当前公开版本，再按对应平台的简短步骤安装。页面下载链接由网站发布配置统一注入。

Windows

便携 ZIP

v1.0.41

适合 Windows 桌面或笔记本。

文件：Tutor1on1.zip 启动：先解压，再运行 tutor1on1.exe
下载 Windows ZIP 查看步骤

Android

直接 APK

v1.0.41

适合 Android 手机或平板。

文件：Tutor1on1.apk 启动：安装 APK 后，从应用列表打开
下载 Android APK 查看步骤

首次使用前

  1. 打开应用，如有需要先切换语言。
  2. 进入设置页，保存 tutor 功能需要的 API 配置。
  3. 在欢迎页登录，或注册老师/学生账号。
