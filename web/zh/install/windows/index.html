Windows

下载 ZIP，解压后运行应用。

Windows 版目前以便携 ZIP 的形式发布，没有单独安装器。

下载 Windows ZIP

安装步骤

  1. 下载 Tutor1on1-1.0.11.zip。
  2. 把 ZIP 解压到普通文件夹，例如 Downloads\Tutor1on1 或 C:\Apps\Tutor1on1。
  3. 打开解压后的文件夹，运行 tutor1on1.exe。
  4. 如果 Windows SmartScreen 弹出提示，请只在你信任该构建来源时继续。

以后如何运行

  • 以后再次运行时，直接打开解压目录中的 tutor1on1.exe。
  • 如果需要桌面或开始菜单快捷方式，可以在解压后手动创建。
  • 不要在 ZIP 查看器里直接运行，先解压。
