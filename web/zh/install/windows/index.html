Windows

下载 ZIP，解压后运行应用。

Windows 版 Tutor1on1 目前以便携 ZIP 发布。先解压再启动，应用才能正常访问内部文件。

下载 Windows ZIP

安装步骤

1

下载

点击上方按钮保存 Tutor1on1.zip。

2

解压

解压到普通文件夹，例如 Downloads\Tutor1on1 或 C:\Apps\Tutor1on1。

3

运行

打开解压后的文件夹，运行 tutor1on1.exe。

注意

  • 如果出现 Windows SmartScreen，请确认来源可信后再继续。
  • 不要直接在 ZIP 预览窗口里运行应用。请先解压。
  • 如果需要快捷方式，可在解压后手动创建。
