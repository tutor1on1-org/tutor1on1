Cómo usar Tutor1on1

Ayuda rápida para docentes y estudiantes.

Tutor1on1 está disponible en Windows, Android y macOS. Los docentes publican cursos estructurados. Los estudiantes se inscriben, descargan contenido aprobado y estudian con sesiones guiadas.

Primeros pasos

  1. Abra la guía de instalación y elija Windows, Android o macOS.
  2. Cambie el idioma si lo necesita.
  3. Abra Ajustes y guarde las API keys necesarias antes de usar las funciones del tutor.
  4. Use la pantalla de bienvenida para iniciar sesión o registrarse como docente o estudiante.

Inicio rápido para docentes

  1. Inicie sesión como docente.
  2. Cree o cargue una versión local del curso.
  3. Suba el bundle validado al marketplace.
  4. Apruebe solicitudes de inscripción y revise el progreso.

Inicio rápido para estudiantes

  1. Inicie sesión como estudiante.
  2. Explore el marketplace y solicite inscripción.
  3. Espere la aprobación y descargue el bundle aprobado.
  4. Abra el curso y comience a aprender o repasar.

Sincronización y problemas comunes

  • Las páginas principales sincronizan en segundo plano cuando la cuenta está vinculada al servidor.
  • El estado de inscripción, las sesiones y el progreso pueden moverse entre dispositivos.
  • Si fallan el inicio de sesión o la carga del curso, primero verifique que la app pueda llegar a api.tutor1on1.org.
