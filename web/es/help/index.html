Inicio rápido y API config

Inicio rápido y API config

Instala la app, inicia sesión como profesor o estudiante y entiende qué controla cada campo de config.

Los profesores crean o cargan cursos, suben bundles validados, aprueban inscripciones y revisan progreso y errores por estudiante.

Los estudiantes entran a cursos aprobados, estudian con Learn y Review, reciben práctica adaptada y usan el cuaderno de errores.

API config

  • Provider: preset del servicio API, como OpenAI o SiliconFlow.
  • Base URL: endpoint compatible con OpenAI. Test API key llama a baseUrl + /models.
  • API key: clave del proveedor guardada localmente en secure storage para ese base URL.
  • Text model: conversación, guía, feedback y evaluación en Review.
  • TTS/STT models: modelos opcionales de voz y reconocimiento.
  • Reasoning effort: ajuste opcional para proveedores compatibles.
  • Test API key: verifica acceso y actualiza modelos. No descarga modelos ni selecciona uno nuevo automáticamente.

Ayuda

Si la lista parece antigua, ejecuta Test API key otra vez. Si un estudiante no entra al curso, revisa aprobación y descarga del bundle.
