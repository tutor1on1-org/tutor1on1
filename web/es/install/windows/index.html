Windows

Descargue el ZIP, extráigalo y luego ejecute la app.

Tutor1on1 en Windows se distribuye actualmente como un ZIP portátil. No hay un instalador separado.

Descargar ZIP de Windows

Pasos de instalación

  1. Descargue Tutor1on1.zip.
  2. Extraiga el ZIP en una carpeta normal como Downloads\Tutor1on1 o C:\Apps\Tutor1on1.
  3. Abra la carpeta extraída y ejecute Tutor1on1.exe.
  4. Si aparece Windows SmartScreen, continúe solo si confía en el origen.

Abrir más tarde

  • Puede volver a abrir la app ejecutando Tutor1on1.exe desde la carpeta extraída.
  • Si quiere un acceso directo, créelo manualmente después de extraer.
  • No la ejecute directamente desde el visor del ZIP.
