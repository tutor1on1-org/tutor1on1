macOS

Descomprima la app, muévala a Applications y luego ábrala.

La versión actual para macOS es un ZIP que contiene Tutor1on1.app. El formato ZIP es el más práctico para la descarga.

Descargar ZIP de macOS

Pasos de instalación

  1. Descargue Tutor1on1-macos-universal.zip.
  2. Haga doble clic en el ZIP para extraer Tutor1on1.app.
  3. Arrastre Tutor1on1.app a /Applications.
  4. Ábrala desde Applications, Launchpad, Spotlight o Finder.

Consejos

  • Si Gatekeeper advierte en el primer inicio, haga clic derecho en la app y luego elija Abrir.
  • Si está instalada en Applications, ábrala con open -a "Tutor1on1".
  • El flujo normal sigue siendo: descomprimir, mover a Applications y ejecutar como app.
