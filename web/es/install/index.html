Elige tu plataforma

Instale Tutor1on1 de la forma correcta para su dispositivo.

Cada plataforma tiene su propio formato de descarga y flujo de apertura. Abra la página adecuada en lugar de adivinar solo por el nombre del archivo.

Windows

ZIP portátil

Descargue el ZIP, extráigalo en una carpeta normal y ejecute family_teacher.exe.

Pasos para Windows
Android

Descarga APK

Descargue el APK, permita la instalación desde esa fuente y luego abra Tutor1on1 desde el lanzador.

Pasos para Android
macOS

ZIP con Tutor1on1.app

Descargue el ZIP, descomprímalo, arrastre Tutor1on1.app a /Applications y luego ábralo normalmente.

Pasos para macOS

Host de descarga

  • Las páginas del sitio se sirven desde www.tutor1on1.org.
  • Los archivos de instalación se sirven desde api.tutor1on1.org/downloads/.
  • El sitio y los artefactos descargables viven en hosts distintos.
